What EPHemeral is
EPHemeral is an advanced wrapper for OpenGL written in and for Java. It uses the LWJGL for native OpenGL bindings. In short:
  • Modern OpenGL rendering
  • Usage of VAOs and VBOs
  • Indexed rendering
  • Shading in GLSL and added language features
  • Interfaces for multi-threaded logic
  • Basic collision using AABB and SAT
  • 2D math and rendering utility
Why write my own boilerplate?
- and not use fantastic libraries like Libgdx
First of all, I enjoy fiddling around with "raw OpenGL" (aside from the wrapper).
Also, writing the essential parts of my projects myself gives me a feeling of control over the code I use and ultimately I get to learn some quite useful hardware rendering skills.
