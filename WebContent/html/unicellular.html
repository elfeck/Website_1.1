What Unicellular is
Unicellular is a toy-project and one huge testsuite for EPHemeral. It is very early in development so there is not much I can tell you right now, apart from beeing 2D. Also, I want to focus on procedural elements instead of texturing.
Here are some features I already implemented
  • 2D movable camera using GLSL uniform offsets
  • Support of different window-resolutions
  • Vertex and fragment based lighting
  • Semirandom generated environment consisting of squares
  • Movable and logicable objects
Some ideas I want to implement in the near future
  • Math-Utility for polygons
  • Advanced procedural shading using Perlin noise
